# Docker Flask PostgreSQL Demo

## Overview

This project demonstrates a simple Flask application connected to a PostgreSQL database.

The purpose of this repository is to learn:

- Docker
- Docker Images
- Docker Containers
- Docker Networking
- Docker Volumes
- Docker Compose
- PostgreSQL
- pgAdmin

---

## Technologies Used

- Python
- Flask
- PostgreSQL
- pgAdmin
- Docker
